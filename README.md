# test-api

## Installation
Clone the repository
```bash
$ git clone git@github.com:hinhengchan/test-api.git
```

Install package
```bash
$ npm install
```

## Run tests
```bash
$ npm test
```

### Parameters
--endpoint: default as http://localhost:51544
```bash
$ npm --endpoint=http://localhost:51544 test
```

--retry: default as 3
```bash
$ npm --retry=3 test
```
